2 factors for determining CSS precedence

  1. Priority based on CSS imports
    1. Inline Style (highest)
    2. Internal Style (higher than external)
    3. External Style - Location of the CSS rule in the file (from top to bottom)

  2. Specificity of the CSS rule

    3. Calculate which is more specific
    1. Count number of ID Selectors 1-0-0
    2. Count number of Class Selectors 0-1-0
    3. Count number of Tag Selectors 0-0-1

  3. !important take over everything (best NOT to use)
